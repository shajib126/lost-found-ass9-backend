# installation and setup

1. Clone this repository ```
    git clone <repositroy_url>
```

2. Install Dependencies : ```
    npm install
```

3. Set up the environment variables by creating a ```
    .env
``` 

4. Run the database migrations ```
npx prisma migrate dev
```

5. Start the server: ```
 npm run start:dev
```
